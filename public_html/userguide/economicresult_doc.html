Financial Result Calender And Table Documentation
In this page we are showing data is two ways in calender and in table format.in both formate we are showing economic event and companies result at that particular day with any market holiday.
Method URL description Page name
JSON API JSON API DOC Returns value of scripts In JSON object later converted to array BlissEventCalender.js
$.post() - Use to fetch data from calender_connect.php. BlissEventCalender.js
createcalender() - Use to create calender BlissEventCalender.js
changedate() - Use to change current date to next or previous date. BlissEventCalender.js
checkevents() - Use to check whether any event is there or not on that day. BlissEventCalender.js
showevents() - Use to show event for that day on tooltip. BlissEventCalender.js

Financial Calender Scenario


  • When BlissEventCalender.php is call it make call to BlissEventCalender.js.
  • In .js file to create financial calender we are using various functions.
  • At first using $.post() function we are fetching data from calender_connect.php to display it in calender.
  • After data is fetch createcalender() function is use to create calender.
  • changedate() function is use to change current date to previous date or next date.using this function we are also changing month and year.
  • checkevents() function is use to check whether any event is there or not at that particular date.
  • showevents() function is use to display event for that particular date.To show events for particular date we are using tooltip,when we place mouse cursor at particular date then if event isthere then at that time events will be display using tooltip.

Financial Table Scenario


  • When BlissEventCalender.php is call it make call to BlissEventCalender.js.
  • In .js file to create financial table we are using datatable api and to know more about it please refer Companiesresulttable_doc.html.
  • At first using $.post() function we are fetching data from calender_connect.php to display it in calender.
  • To provide data to datatable we are getting data from EventCalender_connect.php.
  • In EventCalender_connect.php we are getting data of economic as well as companies result data.
Response


In Calender :-
  • It show companies result or economic event for particular day in calender view.if there is any result or event it is indicated using green color box,while white color circle indicate current date and blue color circle indicate holidays.
In tabular form:-
  • It show same data which is shown in calender in tabular format.here user can see week data together or month data together using filter mechanism
