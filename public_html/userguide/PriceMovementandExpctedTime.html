Price Movement and Expected Time Documentation
Price movement in Companies Result is a movement of stock on Quaterly result day.Movement show range percentage of stock price changes in a day
Expected Time is a time assumption of Quaterly result announcement from historical result time.


Calculation


Attribute formula/ SQL Query Note Page
Movement (High - Low) * 100 / Low percent change of high price with reference
to low price on quarterly result day		 earning_upload.php
Maximum movement SELECT name, Max( Abs( movement ) ) AS maxmovForUser
FROM earning2 where movement <> 0
GROUP BY name
)t
SET max_Movement = ROUND(t.maxmovForUser,1)		 Enable or disable the display of a 'processing' indicator
when the table is being processed		 earning_upload.php
Minimum movement SELECT name, Min( Abs( movement ) ) AS minmovForUser
FROM earning2 where movement <> 0
GROUP BY name
)t SET min_Movement = ROUND(t.minmovForUser,1)		 Enable or disable the display of a 'processing' indicator
when the table is being processed		 earning_upload.php

Response


Attribute Type Note
Price Movement on filter by time period Display max-min movement.
Price Movement on search by company name Display quaterly result day movement.
Expected Time Upcoming result Shows last quarterly result time.
Time Announced result Shows actual result.
Errors


  • Accuracy of expected time is 50%
Scenario


  • Movement show in what range price can move or moved on quarterly result day
  • when we filter the companies result by time period(today,week etc) then it show minimum and maximum movement of stock till now (eg. "1.1 - 4.4" it means from 2013 to current day stock having minimum quarterly result movement is "1.1" and maximum movement is "4.4". it shows this stock have chances to move minimum 1.1% and maximum 4.4%
  • when we search stock name in search bar then table will show result day movement eg: 2.6, it means on that day it moves 2.6%, it shows how much "high value" is greater with respect to "low value" of that day
  • Expected time in companies result is a last quarterly result time. its an assumption that result may come on time of last result.
  • For more accuracy of upcoming time user can search stock and it will show actual time of past result announcement. from all that result time till now user can assume in what time result may come.
