BlissCalc Documentation
BlissCalc is a calculator tool to calculate option greek, profit, loss, etc by creating manual position.BlissCalc is not a webpage, it is an excel application so user can create option strategies manually and calculate its profit/loss, Option Greek etc.

Scenario


  • Enter Spot price,option(call/put) volatility,price, lot size, strike of stratagies and BlissCalc return option greek(Delta, Gamma, Theta, Vega), profit/Loss, margin used, interest, chart, Maximum gain.
  • User have to change days value to calculate profit/loss, option greek for different days left Before expiry
  • Break Even Point can calculate by clicking B.E.P. button
  • Chart Can be show/hide by Show/hide toggle button.
  • Days can be reset if user has changed days, by reset button
  • Save Position by Save As button in Excel 2003 format
  • Clear whole Position by Clear button
