Chart filtering and tooltip Documentation
Filtering mechanism is use to display data day wise,month wise,year wise or any custom range selected by user.this mechanism is use in companies result page,financial calander page,india vix page.

Filtering mechanism


  • Filtering mechanism is created using javascript.
  • In this mechanism by default day is selected.when day is selected data for that particular day is display.
  • If user select week from dropdown then data of last seven days from current date is display in table.
  • When week is selected firstly we are fetching current system date and store it in one variable and to get 7 th day date we are subtracting 7 from current date.
  • same process is done when month is selected we subtract 30 from current date and when year is selected we are adding 1 to year.
  • when user click this "<<" we are showing previous data according to the selected range.if week is selected then previous week ,if month then previous month etc
  • When user click this ">>" we are showing next data according to the selected range.if week is selected then next week,if month then next month etc.
  • When user enter custom range then for that particular period data is display.

Tooltip


  • Tooltip is use to display information on mouse hover at particular position on chart.
  • Using tooltip we are displaying OHLC information with date,previous change,close info also.
  • For tooltip generateChartData() Function is use.
  • In this function first we are getting data from ohlc_connect.php in JSON object.
  • Then this data is store in variables and this variables are display useing "Baloon Text" in chart.
