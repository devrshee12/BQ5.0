Response


In BlissIndiaVix.php :-
  • It show comparision between IndiaVIX and Nifty,Where blue line show IndiaVIX and green line show Nifty while white line show more than 1% up or down in market
In BlissCboeVix.php:-
  • It show comparision between IndiaVIX and CBOE VIX,Where blue line show IndiaVIX and green line show CBOE VIX while white line show more than 10% up or down in market
Errors


  • When more or less parameter than required is pass then it will show blank screen but no error will be display.
  • If website is open in internet explorer then also it display blank screen instead of chart.
Scenario


  • When BlissIndiaVix.php or BlissCboeVix.php is called,it make call to BlissIndiaVix2.js or cboe_vix.js.
  • In .js file we are getting data from Database(Tables like:-cboevix,india_vix).
  • To get data from database we are calling iv_connect or cboe_vix_connect.php.
  • From this file we are getting data in form of JSON object
  • And this data is pass to amchart api to show chart.
