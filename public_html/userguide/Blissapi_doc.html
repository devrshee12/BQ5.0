Google API http://finance.google.com/finance/info?client=ig&q=indeces name. Google API is use to take live price of indeces.value for indeces comes from url in form of file and then it is converted to json object and later in array.Eg:-http://finance.google.com/finance/info?client=ig&q=INDEXBOM:sensex returns value of sensex.
Yahoo API http://download.finance.yahoo.com/d/quotes.csv?s=$quote&f=l1t1p2c1&e=.csv Yahoo API is use to take live prices of indeces.in url we are passing various parameter to get content in file format.To know more about parameter refer this link.Yahoo API Documentation
BootStrap3 bootstrap api bootstrap framework is use for UI.we are mainly useing two files "bootstrap.min.css","bootstrap.js".refer link to know about bootstrap.
