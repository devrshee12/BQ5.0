Database Connection Documentation
This documentation explain database connectivity of mysql with php and functions used in connection.

Method Type description Page name
mysqli_connect boolean Open a new connection to the MySQL server. db_connect.php
mysqli_select_db boolean Selects the default database to be used when performing queries against the database. db_connect.php
mysqli_set_charset boolean Sets the default client character set. db_connect.php
mysqli_query mixed Performs a SQL queries(insert,update,delete...) on the database. All connection pages
mysqli_num_rows integer Retrieves the number of rows from a result set. All connection pages
mysqli_fetch_row mixed Get a result row as an enumerated array All connection pages
mysqli_close boolean Closes a previously opened database connection. All connection pages

mysqli_connect Response Parameter


Name Note Required? Default
host Can be a host name or IP address Required
username The MySQL user name Required
Password MySQL authentication password Required
database Default database to be used when performing queries Optional
Port Specifies port number to attempt to connect to the MySQL server. Optional 3306
socket Specify the socket or named pipe that should be used Optional TCP

Scenario


  • open a new connection by mysqli_connect by passing server name, username and password and then select database by mysqli_select_db
  • select character set utf8 by mysqli_set_charset.
  • run mysql queries by mysqli_query function by passing queries in function
  • get the number of rows by mysql_num_rows and fetch the row data one by one by mysql_fetch_row function, by passing object return from mysqli_query function
  • after receiving all data, close mysql connection by mysqli_close function
