News Ticker Documentation
In Vertical News Ticker :-
  • Live price of Sensex , Nifty , BankNifty and India VIX , Gainers and Loosers (Scripts with +/- 5% movement)
  • 52 week high and low , Life time high and low , Security in ban period script
  • Companies result(show name of company having result at that day) , Economic Event(show event name for that day)
In Horizontal News Ticker :-
  • Live price of Sensex , Nifty , BankNifty , India VIX and Other Sector wise Price, Gainers and Loosers (Scripts with +/- 5% movement)
  • Companies result(show name of company having result at that day) , Economic Event(show event name for that day) , Security in ban period script

Errors


  • Errors occurs in one of this scenario:
    • Internet connection is lost or server pc is not working then error for database connection occur
    • If text file is missing
Format


  • When data is coming from text file it comes in form of array
  • When data is coming from google API it comes in JSON object later it is converted in array
Scenario


  • When news_ticker_footer.php is called in every page,it make call to newsticker_read.php.
  • In newsticker_read.php we are getting data from Database,Google API and text file
  • To get data from database query is fire to MySQL Database which gives data from relevant table in form of array which is later split in to string.
  • To get data from text file we are using fopen function to open file then fgets() function to get data from text file in array.
  • To get data from Google API we first pass script name to Google API which in return provide data for that script in form of JSON objet which is converted in to array.
